# Фаза 4: Интеграция SD-карты (exFAT) и Логирование

## Обзор и Цели
В соответствии с требованиями архитектуры, **Фаза 4** выделена под критически важный функционал: надежное журналирование (логирование) всего принимаемого текста на внешнюю MicroSD-карту в файловой системе **exFAT**. 
Поскольку декодер работает в жестком реальном времени (Hard Real-Time), запись на карту не должна вызывать блокировок или потери сэмплов аудиосигнала.

## 1. Аппаратное подключение
- **SPI Шина:** SD-карта будет подключена к выделенному аппаратному SPI-контроллеру (например, `spi1`), чтобы не конфликтовать с высокоскоростной шиной дисплея (`spi0`, 62.5 МГц).
- **Пины (Примерные):** MISO, MOSI, SCK, CS. Будут уточнены на этапе пайки модуля кардридера.

## 2. Программная архитектура (FreeRTOS / FatFs)
- **Библиотека:** Использование портированной для RP2350 библиотеки `FatFs` с обязательной поддержкой длинных имен файлов (LFN) и **exFAT** (для поддержки современных карт памяти объемом от 64 ГБ).
- **Кольцевой буфер (Ring Buffer):** 
  Ядро 0 (или Ядро 1) декодирует символ и мгновенно (без блокировки) кладет его в неблокирующий кольцевой буфер (Lock-free FIFO queue) размером 4-8 Килобайт.
- **Фоновая запись:**
  Отдельная низкоприоритетная задача (или прерывание таймера на Ядре 1) периодически проверяет буфер. Если накопилось достаточное количество данных (например, 512 байт, размер сектора), данные сбрасываются на карту памяти (`f_write`, `f_sync`).
- **Синхронизация:** Использование аппаратных мьютексов (`hardware_sync`) или Spin-locks для безопасного доступа к буферу с разных ядер.

## 3. Формат Логирования
Текст будет сохраняться в текстовые файлы `.txt` с временными метками (если в будущем будет добавлен модуль RTC) или авто-инкрементными именами:
- Файлы: `LOG_0001.TXT`, `LOG_0002.TXT` и т.д.
- В будущем: разбор DWD SYNOP метеосводок и сохранение их в отдельные структурированные таблицы `.csv`.

## 4. UI и Управление
- В меню появится кнопка `SD: RECORD` / `SD: STOP`.
- Индикатор на главном экране (например, красная точка или иконка дискеты), показывающий активный процесс записи.
- Предупреждения об ошибках: `SD ERROR`, `SD FULL`.

*Примечание: Разработка этой фазы отложена до физического подключения модуля SD-кардридера к плате.*
