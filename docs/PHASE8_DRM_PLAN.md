# Фаза 8: SDR Панорама и DRM Декодер (HE-AAC)

Этот документ описывает стратегический план трансформации TouchRTTY в полноценный SDR-приемник, способный принимать и декодировать цифровое радиовещание DRM (Digital Radio Mondiale).

## 1. Концепция и входные данные
Использование приемника **Belka-DX** в качестве RF-фронтенда.
- **Интерфейс:** IQ выход (стерео-джек).
- **Полоса пропускания:** 40 кГц.
- **Метод ввода:** Двухканальное сэмплирование через ADC0 и ADC1 микроконтроллера RP2350.

## 2. Техническое задание (ТЗ)

### 2.1. Низкоуровневая часть (Core 0 - SDR Front-end)
- [ ] **Dual ADC DMA:** Настройка одновременного сэмплирования двух каналов (I и Q) на частоте 96 кГц.
- [ ] **Complex Filter:** Реализация программного подавления зеркального канала (Hilbert Transform / Complex Mixer).
- [ ] **Digital Down-Conversion (DDC):** Перенос выбранного участка спектра (10 кГц для DRM) в baseband.
- [ ] **OFDM Demodulation:** Синхронизация по поднесущим, снятие квадратурной модуляции (QAM-16/64).

### 2.2. Аудио и Данные (Core 1 - Decoder & UI)
- [ ] **AAC Decoding:** Портирование библиотеки **FDK AAC (Fixed-Point)** или **Helix AAC** для декодирования аудиопотока в реальном времени.
- [ ] **I2S Audio Output:** Реализация драйвера для внешнего ЦАП (например, PCM5102) через протокол I2S.
- [ ] **DRM Data Parser:** Декодирование текстовых новостей (Journaline) и служебной информации.
- [ ] **Wideband Waterfall:** Отрисовка панорамы всех 40 кГц входного сигнала.

## 3. Оценка ресурсов RP2350

| Ресурс | Требуется для DRM | Доступно на RP2350 | Запас |
| :--- | :--- | :--- | :--- |
| **CPU Speed** | ~150-200 MHz | 300 MHz | 30% |
| **RAM** | ~250 KB (буферы + таблицы) | 520 KB | 50% |
| **FPU** | Высокая нагрузка (float) | Аппаратный (Single Precision) | OK |
| **Hardware** | I2S Output | PIO / PWM | Требуется внешний ЦАП |

## 4. Этапы реализации
1. **Этап А:** Реализация двухканального АЦП и визуализация 40 кГц спектра (Панорама).
2. **Этап Б:** Подключение внешнего ЦАП и вывод обычного AM/SSB звука через RP2350.
3. **Этап В:** Внедрение OFDM демодулятора.
4. **Этап Г:** Интеграция AAC-кодека и финальное тестирование приема DRM станций (например, DWD или Radio Romania Intl).
