# TouchRTTY: Master Session Context (Save State)
**Дата последнего обновления:** 2026-04-04
**Текущая ветка:** `feature/rtty-dsp-improvements`
**Текущий билд:** 205 (RC3 UI + Hardware ADC FIFO)

## 1. ТЕКУЩИЙ СТАТУС ПРОЕКТА
- **DSP (Build 205):** Внедрен аппаратный FIFO буфер АЦП с жестким тактированием 10кГц. Это полностью устранило джиттер выборки и кардинально улучшило стабильность DPLL. FFT вычисляется на Core 0.
- **Интерфейс:** Перенесены все наработки из RC3: индикатор уровня ошибок (ERR) с прогресс-баром, всплывающее окно настройки частот MARK/SPACE, компактная стопка баров SIG/AGC/ERR.
- **Функции:** Защита от deadlock при сбросе через энкодер на старте системы. Ограниченное (rolling) окно подсчета ошибок. Исправлен баг перекрытия всплывающих окон.

## 2. ИНЖЕНЕРНЫЕ ПРАВИЛА (ОБЯЗАТЕЛЬНО К ИСПОЛНЕНИЮ)
1. **Surgical Edits Only:** Не перезаписывать `main.cpp` целиком.
2. **Strict Float Policy:** ТОЛЬКО `float` константы (`f`) и функции (`sinf()`, `cosf()`).
3. **RAM Execution:** DSP-функции с атрибутом `__time_critical_func`.
4. **Modulo Removal:** Запрещено использовать `%` в горячих циклах Core 0.
