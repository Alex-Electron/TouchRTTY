## 2. ИНЖЕНЕРНЫЕ ПРАВИЛА (ОБЯЗАТЕЛЬНО К ИСПОЛНЕНИЮ)
1. **Surgical Edits Only:** Не перезаписывать `main.cpp` целиком. Использовать `replace` для точечных правок.
2. **Strict Float Policy:** Использовать ТОЛЬКО `float` константы (суффикс `f`, напр. `1.65f`) и функции `sinf()`, `cosf()`, `fabsf()`. Никаких `double`.
3. **RAM Execution:** Помечать DSP-функции атрибутом `__time_critical_func`.
4. **Modulo Removal:** Запрещено использовать `%` в горячих циклах Core 0.
5. **Memory Barriers:** Использовать `__dmb()` при передаче данных между ядрами (FFT буфер).
6. **Wait For Event:** Использовать `__wfe()` для разгрузки шины при ожидании данных АЦП.
